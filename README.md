# 📊 Java Attendance Manager

A desktop-based attendance management system built using Java Swing.
This project allows basic student management and attendance tracking with a simple graphical interface. It focuses on combining core Java concepts like OOP, file handling, and GUI development into a single working application.

---

## 📌 Features

- 👤 Add new students (Roll No + Name)
- ✅ Mark attendance (Present / Absent)
- 📈 Live attendance report with percentage
- 📊 Automatic sorting based on attendance percentage
- 🔄 Reset attendance for all students
- 💾 Persistent storage using file handling (`students.txt`)
- 📤 Export attendance report as a `.txt` file
- 🖥️ Simple GUI using Java Swing

---

## ⚙️ How to Run

Make sure Java is installed on your system.

### Compile:
javac AttendanceManager.java
### Run:
java AttendanceManager
